Airquality analysis

This is a quick demo of a Quarto website where we’ll present airquality data. To learn more about Quarto websites visit https://quarto.org/docs/websites.

Air Quality

Figure 1 further explores the impact of temperature on ozone level.

Code 
library(ggplot2)
ggplot(airquality, aes(Temp, Ozone)) + 
  geom_point() + 
  geom_smooth(method = "loess")

The data for this analysis are shown below using the DT package in R, an interface to javascript datatables.

Code 
library(DT)
datatable(airquality)
